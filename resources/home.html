Технологии разработки приложений для мобильных платформ



Основные этапы реализации проекта:

- Реализована установка приложения через собственный интерфейс, кнопку INSTALL (A2HS);

- Кнопка появляется вместо собственного предложения браузера (Chrome) об установке на рабочий стол. С этой целью перехватывается событие beforeinstallprompt;

- После установки, при запуске мобильного приложения с рабочего стола кнопка INSTALL исчезает;

- Приложение реализовывает кэширование всех своих активов посредсвом ServiceWorker;

- В одной из вкладок приложение показывает Emojis, предоставляемые GitHub REST API (https://api.github.com/emojis);

- Реализована "умную" загрузку изображений, используя технологию Progressive Image Loading and IntersectionObserver;

- Реализована "распечатка" журнала сообщений от ServiceWorkera

- сертификат (скачать), подписанный собственным CA
